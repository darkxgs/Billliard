# Royal Eight — Billiards

A premium 3D Billiards game built using Three.js, GSAP, and TypeScript.

## Features
- Realistic 3D table, cue stick, and ball physics (sliding, rolling, cushion impact, English/spin).
- Smooth orbit camera controls (drag to aim, pinch to zoom).
- Dual player turn-based 8-ball rules.
- Immersive audio effects and visual particle bursts.

## Tech Stack
- **Core**: HTML5, TypeScript
- **Graphics & Rendering**: Three.js
- **Animations**: GSAP
- **Bundler**: Vite
